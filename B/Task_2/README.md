# SkillMatch – Requirements Engineering Project

## 📌 Overview

**SkillMatch** is a conceptual platform designed to connect job seekers with relevant job opportunities based on their skills, experience, and preferences.
The goal of this project is to demonstrate structured **requirements engineering practices**, including requirement definition, organization, validation, and iterative development.

This project does **not include implementation**, but focuses on the **analysis, design, and documentation** of a software system.

---

## 🎯 Objectives

* Define clear and testable requirements
* Organize requirements using both simple and professional tools
* Establish a project constitution (vision, mission, principles)
* Plan system development through multiple iterations
* Validate requirements through structured test cases

## 🧩 Project Components

### 1. Requirements

* A set of functional and non-functional requirements
* Organized in:

  * A **simple tool**
  * A **professional structure** with detailed attributes

---

### 2. Constitution

Defines the foundation of the project, including:

* Vision
* Mission
* Core values
* Guiding principles

---

### 3. Iterations

The system is developed conceptually in multiple iterations:

* **Iteration 1:** Basic functionality (profiles, job listings)
* **Iteration 2:** Matching and ranking system
* **Iteration 3:** Advanced features (AI recommendations, notifications)

Each iteration includes:

* Specification document
* Validation document
