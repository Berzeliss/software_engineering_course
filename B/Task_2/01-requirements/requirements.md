# Requirements

## Requirements List

### Functional Requirements

**REQ-01** The system needs to allow the user to create/edit/delete a profile.

**REQ-02** The system needs to allow the user to add/update/remove skills/education/experience.

**REQ-03** The system needs to allow companies to create/edit/delete a job listing.

**REQ-04** The system needs to match job listings to profiles based on skills and experience.

**REQ-05** The system needs to allow users to apply for a job directly through the platform.

### Non-Functional Requirements

**REQ-06** The system needs to respond to user queries under 5 seconds.

**REQ-07** The system needs to ensure that all user data is protected from unauthorised access.

**REQ-08** The system needs to be available 99% of the time, only being down for scheduled maintenance.

**REQ-09** The system needs to be able to handle at least 1000 users concurrently.
